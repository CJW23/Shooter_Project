# Shooter_Project
Unity,KinectV2,EyeTracking의모션인식과,동공인식을 이용한 슈팅 게임.

# Description
- 동공인식과 모션인식을 이용하여 다가오는 몬스터들을 처치하는 게임 프로젝트이다.
- EyeTracking의 동공 인식을 통해 적을 바라봄으로써 조준이 되고, Kinect의 모션 인식을 통해 총알을 발사할 수 있다.
- 손을 위로 올려 총을 발사 할 수 있다.
